## PCB

---

This pad will be handwired. A custom PCB will not be designed for two primary reasons:

- A 100×100 mm PCB costs approximately $16–$20 including shipping. For a simple 3×3 pad, this cost is disproportionate to the project’s complexity.
- The circuit is straightforward and does not justify PCB fabrication. The pad consists of only a few components and does not require dense routing or multilayer design.
